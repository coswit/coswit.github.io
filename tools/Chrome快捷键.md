**标签页和窗口**

- Ctrl + T: 打开新标签页
- Ctrl + W: 关闭当前标签页
- Ctrl + Shift + T: 重新打开最近关闭的标签页
- Ctrl + Tab: 切换到下一个标签页
- Ctrl + Shift + Tab: 切换到上一个标签页
- Ctrl + 1~8: 跳转到指定编号的标签页
- Ctrl + 9: 跳转到最后一个标签页
- Ctrl + N: 打开新窗口
- Ctrl + Shift + N: 打开无痕模式窗口

**导航**

- Alt + ←: 后退
- Alt + →: 前进
- F5 或 Ctrl + R: 刷新页面
- Ctrl + Shift + R: 强制刷新(清除缓存)
- Esc: 停止加载页面
- Ctrl + L 或 Alt + D: 跳转到地址栏

**页面操作**

- Ctrl + F: 在页面中查找
- Ctrl + P: 打印
- Ctrl + S: 保存网页
- Ctrl + D: 将当前页面加入书签
- Ctrl + Shift + D: 将所有打开的标签页加入书签
- Ctrl + +: 放大页面
- Ctrl + -: 缩小页面
- Ctrl + 0: 重置缩放

**页面滚动**

- Space(空格键): 向下滚动一屏
- Shift + Space: 向上滚动一屏
- Home: 滚动到页面顶部
- End: 滚动到页面底部
- Page Down(PgDn): 向下滚动一屏
- Page Up(PgUp): 向上滚动一屏
- ↑/↓ 方向键: 向上/向下小幅滚动
- ←/→ 方向键: 向左/向右滚动(适用于有横向滚动条的页面)

**其他常用**

- Ctrl + H: 打开历史记录
- Ctrl + J: 打开下载内容
- Ctrl + Shift + Delete: 清除浏览数据
- F12 或 Ctrl + Shift + I: 打开开发者工具
- Ctrl + Shift + B: 显示/隐藏书签栏
