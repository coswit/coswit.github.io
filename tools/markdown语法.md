####  类图classDiagram

- 类、方法、返回值定义

 `+` Public
 `-` Private
 `#` Protected
 `~` Package/Internal

 `*` Abstract e.g.: `someAbstractMethod()*`
 `$` Static e.g.: `someStaticMethod()$`

显示

classDiagram

          +swim()
